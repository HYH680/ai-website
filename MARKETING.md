## 1. 产品基础设置

### 1.1 上线前检查清单

- [ ] 部署到 Vercel（`vercel --prod`）
- [ ] 配置环境变量：
  - `OPENAI_API_KEY` — OpenAI API 密钥
  - `JWT_SECRET` — 随机 32 位字符串
  - `DATABASE_URL` — Vercel Postgres 连接串
  - `SITE_URL` — 你的域名
- [ ] 绑定自定义域名
- [ ] 配置 HTTPS（Vercel 自动）
- [ ] 提交到百度搜索资源平台
- [ ] 提交到 Google Search Console
- [ ] 生成 sitemap.xml
- [ ] 配置百度统计 / GA4

### 1.2 域名建议

| 类型 | 示例 | 备注 |
|------|------|------|
| .cn 域名 | ai-zhushou.cn | 国内推荐，对 SEO 友好 |
| .com 域名 | myaibot.com | 国际化 |
| .top 域名 | aibest.top | 便宜，但不是最优 |
| 二级域名 | ai.yourblog.com | 零成本起步 |

### 1.3 需补充的页面

在部署前或部署后尽快添加：

1. **`/privacy`** — 隐私政策
2. **`/terms`** — 服务条款
3. **`/about`** — 关于我们
4. **`/blog`** — 博客/教程区（SEO 核心）
5. **`/sitemap.xml`** — 站点地图
